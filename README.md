# Input Masks Pratice  

> **This project I pratice ReactJS making a masked input Component**  

## Runing the project

> **Pré-conditions**  
>
> YARN and NPM

- Clone this repository:  
`git clone http://github.com/cezarcozta/inputMask.git`  

- Install the dependencies:  
`yarn`  

- Run the App  
`yarn start`  

- See the broweser on:  
[http://localhost:3000]
